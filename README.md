Motion Detection Web App Presentation

This repository contains a presentation demonstrating a motion detection web application built with JavaScript, HTML, and CSS. The presentation uses screenshots from the final demo to illustrate key functionality, including:

Real-time movement detection via webcam input
Visual alerts triggered by motion
Event logging and playback

This project highlights problem-solving, debugging, and event-driven programming skills in a web development context.

Note: This repo contains only the presentation/demo screenshots, not the full working application.
